The four games shown above are games that focus on climate change, the environment and how to be more eco friendly.


The first game is hang man, the goal is to guess the word before the image is complete to win.


The second game is a quiz that will improve your knowledge on the environment and climate change.


The third game is flashcards to set up your own questions and test yourself to help memorize information on these subjects.


Lastly, the fourth game is a word scramble. This is a more fun and interactive way to help remember everything.
